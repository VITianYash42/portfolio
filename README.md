# yashbuilds.xyz

My personal developer portfolio built with Next.js, Three.js, and Framer Motion.

## Tech Stack

- **Framework** — Next.js 15 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS
- **3D** — Three.js + React Three Fiber
- **Animations** — Framer Motion
- **Deployment** — Vercel

## Features

- Interactive 3D particle network background with custom shaders
- Mouse-tracking spotlight cards with 3D tilt effect
- Magnetic skill pills with spring animations
- Animated terminal CLI component
- Custom neon cursor with trailing effect
- Scroll-triggered section animations
- Fully responsive design

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed on [Vercel](https://vercel.com). Every push to `main` triggers an automatic production deployment.
